# _api/songs/
> L2 | 父级: /_api/CLAUDE.md

## 成员清单

index.ts: 歌曲列表端点，获取用户歌曲库、搜索歌曲、导入 YouTube
[id].ts: 歌曲详情端点，获取歌曲元数据、歌词、翻译、注音
_constants.ts: 歌曲常量，歌词源 API、缓存配置、超时时间
_furigana.ts: 日语注音工具，假名转换、汉字注音、罗马音
_kugou.ts: 酷狗歌词 API，搜索歌词、下载 LRC、解析时间轴
_lyrics.ts: 歌词处理工具，LRC 解析、时间轴同步、翻译合并
_soramimi.ts: 空耳歌词工具，音译歌词、谐音歌词
_utils.ts: 歌曲工具函数，元数据提取、封面获取、时长格式化

## 依赖关系
- 依赖 YouTube API 获取元数据
- 依赖酷狗 API 获取歌词
- 依赖 Redis 缓存歌词和元数据
- 被前端 iPod 应用调用

## 歌曲约束
1. 歌曲 ID 使用 YouTube Video ID
2. 歌词必须缓存到 Redis，过期时间 7 天
3. 歌词搜索支持模糊匹配、艺术家过滤
4. 支持多语言歌词（原文、翻译、音译）
5. 歌词时间轴精度到毫秒
6. 支持日语假名注音、韩语罗马音
7. 元数据包含标题、艺术家、专辑、封面、时长
