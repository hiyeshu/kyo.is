# _api/auth/token/
> L2 | 父级: /_api/auth/CLAUDE.md

## 成员清单

refresh.ts: 令牌刷新端点，延长令牌有效期，避免频繁登录
verify.ts: 令牌验证端点，检查令牌是否有效，返回用户信息

## 依赖关系
- 依赖 _utils/auth 认证工具
- 依赖 Redis 存储令牌
- 被前端认证中间件调用

## 令牌管理约束
1. 令牌刷新必须验证旧令牌有效性
2. 刷新后旧令牌立即失效
3. 令牌验证失败返回 401 状态码
4. 令牌过期时间默认 30 天
5. 令牌必须包含用户名、创建时间、设备信息
6. 令牌验证必须有速率限制
7. 令牌存储到 Redis，键格式：auth:token:{token}
