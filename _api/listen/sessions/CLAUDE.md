# _api/listen/sessions/
> L2 | 父级: /_api/listen/CLAUDE.md

## 成员清单

index.ts: 会话列表端点，获取用户的监听会话列表、创建新会话
[id]/ - 会话操作端点，获取会话详情、更新进度、结束会话

## 依赖关系
- 依赖 Redis 存储会话数据
- 依赖 Pusher 实时同步
- 依赖 _api/songs 歌曲元数据
- 被前端 iPod 应用调用

## 会话管理约束
1. 会话 ID 使用 UUID 生成
2. 会话状态包含：歌曲 ID、进度、播放状态、参与者
3. 会话过期时间 1 小时，自动清理
4. 进度更新必须实时广播到所有参与者
5. 会话结束时清理 Redis 数据和 Pusher 频道
6. 支持暂停、播放、跳转操作
7. 参与者列表实时更新
